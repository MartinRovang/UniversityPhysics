How to start the game (Windows):
Run python main.py in the command console.


Introduction:

This project is a clone of the game "Breakout". Play as a platform and bounce the ball to break the bricks at the top of your screen. 
If you break all the bricks you win the game, but if the ball drops below the screen you lose.

Controls:

You control the platform with the left and right key on your keyboard. 
The ball will bounce of in an angle depending on where you hit on the platform.
The further out on the edges of the platform the ball hits the broader the angle.
